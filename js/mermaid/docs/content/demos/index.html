Loops, alt and opt

%% Sequence diagram code
sequenceDiagram
	loop Daily query
		Alice-&gt;&gt;Bob: Hello Bob, how are you?
		alt is sick
			Bob-&gt;&gt;Alice: Not so good :(
		else is well
			Bob-&gt;&gt;Alice: Feeling fresh like a daisy
		end

		opt Extra response
			Bob-&gt;&gt;Alice: Thanks for asking
		end
	end

		end
